# 🛡️ SecureSocial - Защита аккаунтов социальных сетей

Пошаговые инструкции по безопасности ВКонтакте, Telegram и TikTok. Учебный проект по кибербезопасности с подробными скриншотами и советами.

## 📋 Описание

**SecureSocial** помогает пользователям защитить свои аккаунты в социальных сетях за 15 минут. Проект включает готовые инструкции для трех популярных платформ: ВКонтакте, Telegram и TikTok, с адаптацией под мобильные и десктопные версии. 

## � Ветка main

На удалённом репозитории `main` должна содержать точно эту рабочую версию проекта: текущие HTML-страницы, схема `assets/`, скрипты, стили и изображения. Все устаревшие файлы, дополнительные ветки и ненужная история не должны присутствовать в `main`.

> Эта ветка — единственный актуальный снимок проекта. Если вы работаете с этим репозиторием локально, синхронизируйте с `origin/main`, чтобы получить ту же самую версию.

## �🚀 Запуск проекта

### ✅ Самый простой способ (для всех)

1. **Найдите папку с проектом**
   - Откройте папку `CyberShield` на вашем компьютере

2. **Откройте главную страницу**
   - Найдите файл `index.html`
   - Нажмите на него **правой кнопкой мыши**
   - Выберите **«Открыть с помощью» → ваш браузер** (Chrome, Firefox, Opera и т.д.)

3. **Готово!**
   - Сайт откроется в браузере и будет работать полностью

> **Что происходит?** Все файлы проекта работают локально (на вашем компьютере), не нужен интернет. Это абсолютно безопасно и быстро.

## 📁 Точная структура проекта (рабочая копия)

В репозитории содержится готовая локальная статическая версия сайта. Ниже — точная структура файлов и папок, как они находятся в этой копии проекта:

```
CyberShield/
├── index.html               # Главная страница
├── instructions.html        # Как пользоваться сайтом
├── vksecure.html            # Пошаговая инструкция — ВКонтакте
├── tgSecure.html            # Пошаговая инструкция — Telegram
├── ttSecure.html            # Пошаговая инструкция — TikTok
├── vk-checklist.html        # Чек-лист — ВКонтакте
├── tg-checklist.html        # Чек-лист — Telegram
├── tt-checklist.html        # Чек-лист — TikTok
├── quiz.html                # Викторина по безопасности
├── password-tester.html     # Тестер пароля
├── README.md                # Этот файл (описание + инструкция)
├── .gitignore               # Игнорируемые файлы
└── assets/
   ├── css/
   │   └── styles.css      # Основные стили
   ├── js/
   │   └── common.js       # Общая логика (чек-лист, lightbox, тесты)
   └── images/
      ├── vk/             # Скриншоты для VK (step1..step7/)
      ├── tg/             # Скриншоты для Telegram (step1..step7/)
      └── tt/             # Скриншоты для TikTok (step1..step4/)
```

Примечание: в рабочей копии все ссылки в HTML указывают на файлы внутри `assets/`.

## 💻 Использование

### Главная страница (index.html)
1. Выберите социальную сеть (ВКонтакте, Telegram или TikTok).
2. Перейдите на страницу с подробной инструкцией по защите аккаунта.

### Страница инструкции по использованию (instructions.html)
1. Узнайте, как пользоваться сайтом: выбор соцсети, чек-лист, тестер пароля и викторина.
2. Это удобный старт для новых пользователей.

### Страницы инструкций (vksecure.html, tgSecure.html, ttSecure.html)
1. **Переключение режимов**: используйте кнопки "📱 Телефон" и "💻 ПК", чтобы увидеть скриншоты для нужного устройства.
2. **Следуйте инструкциям**: каждый шаг содержит:
   - название и описание
   - путь к нужному пункту меню
   - скриншоты с разметкой
   - советы и рекомендации
3. **Чек-лист**: отмечайте выполненные пункты, прогресс сохраняется автоматически.

### Тестер пароля (password-tester.html)
1. Введите свой пароль или нажмите «Сгенерировать».
2. Следите за оценкой силы пароля и выполните предложенные критерии.
3. Используйте подсказки для создания более надёжного пароля.

### Викторина (quiz.html)
1. Перейдите на страницу викторины через кнопку в шапке.
2. Пройдите три интерактивных задания по безопасности.
3. После выполнения всех заданий появится общий результат.
