# 🛡️ SecureSocial - Защита аккаунтов социальных сетей

Пошаговые инструкции по безопасности ВКонтакте, Telegram и TikTok. Учебный проект по кибербезопасности с подробными скриншотами и советами.

## 📋 Описание

**SecureSocial** помогает пользователям защитить свои аккаунты в социальных сетях за 15 минут. 

## 🚀 Запуск проекта

### ✅ Самый простой способ (для всех)

1. **Найдите папку с проектом**
   - Откройте папку `CyberShield` на вашем компьютере

2. **Откройте главную страницу**
   - Найдите файл `index.html`
   - Нажмите на него **правой кнопкой мыши**
   - Выберите **«Открыть с помощью» → ваш браузер** (Chrome, Firefox, Opera и т.д.)

3. **Готово!**
   - Сайт откроется в браузере и будет работать полностью

> **Что происходит?** Все файлы проекта работают локально (на вашем компьютере), не нужен интернет. Это абсолютно безопасно и быстро.

---

### 🚀 Продвинутый способ (если хочется через командную строку)

Этот способ нужен, если у вас возникнут проблемы с первым методом.

<details>
<summary><b>Запуск через Windows (самое простое)</b></summary>

1. **Откройте папку проекта**
   - Найдите папку `CyberShield`
   - Нажмите `Ctrl + L` чтобы увидеть адрес (или кликните на адресную строку)
   - Скопируйте полный путь (например: `D:\CyberShield`)

2. **Откройте PowerShell (командная строка)**
   - Нажмите **Win + X**, выберите **Windows PowerShell**
   - Или найдите **PowerShell** в меню Пуск

3. **Перейдите в папку проекта**
   ```
   cd D:\CyberShield
   ```
   (замените `D:\CyberShield` на ваш путь из шага 1)

4. **Запустите простой сервер**
   
   **Вариант A (Python - самый простой):**
   ```
   python -m http.server 8000
   ```
   
   **Вариант B (если Python не установлен - используйте PHP):**
   ```
   php -S localhost:8000
   ```

5. **Откройте браузер**
   - Введите в адресную строку: `http://localhost:8000`
   - Нажмите Enter

6. **Остановить сервер**
   - В PowerShell нажмите `Ctrl + C`

</details>

<details>
<summary><b>Запуск через Mac/Linux</b></summary>

1. **Откройте Терминал**
   - Mac: Cmd + Space, введите `Terminal`
   - Linux: Ctrl + Alt + T

2. **Перейдите в папку проекта**
   ```
   cd /path/to/CyberShield
   ```
   (замените на реальный путь к папке)

3. **Запустите сервер (выберите один вариант)**
   
   **Вариант A (Python):**
   ```
   python -m http.server 8000
   ```
   
   **Вариант B (если Python не работает):**
   ```
   php -S localhost:8000
   ```

4. **Откройте браузер**
   - Введите: `http://localhost:8000`

5. **Остановить сервер**
   - Нажмите `Ctrl + C`

</details>

### 📌 Если ничего не работает

**Проблема:** Файл не открывается двойным кликом

**Решение:** 
- Откройте браузер (Chrome, Firefox и т.д.)
- Нажмите `Ctrl + O` (Cmd + O на Mac)
- Выберите файл `index.html` из папки проекта

## 💻 Использование

### Главная страница (index.html)
1. Выберите социальную сеть (ВКонтакте, Telegram или TikTok)
2. Вас перенесет на страницу с инструкциями

### Страница ВКонтакте (vkSecure.html)
1. **Переключение режимов**: Используйте кнопки "📱 Телефон" и "💻 ПК" чтобы видеть скриншоты для нужного устройства
2. **Навигация**: Нажимайте на номера шагов в верхней панели для быстрого перехода
3. **Следуйте инструкциям**: Каждый шаг содержит:
   - Название и описание
   - Путь к нужному пункту меню
   - Скориншоты с разметкой
   - Советы и рекомендации
